# .dotfiles

## install

```zsh
# dotfiles
git clone --recursive https://github.com/iamvlado/.dotfiles && ~/.dotfiles/init

# Vim's things
git clone https://github.com/gmarik/vundle.git ~/.vim/bundle/vundle

# Oh My Zsh
sh -c "$(curl -fsSL
https://raw.github.com/robbyrussell/oh-my-zsh/master/tools/install.sh)"
```
